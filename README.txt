单词记忆轻程序（iPhone 稳定版 v3）

这版保留原来的 7 个界面：
- 今日
- 复习
- 录入
- 日历
- 词库
- 记录
- 设置

本次新增/调整：
1. 词库支持编辑、删除
2. 日历支持：
   - 年份/月筛选
   - 上个月 / 下个月
   - 查看当天录入单词
   - 补录到该日期
   - 编辑 / 删除
   - 从该日批次进入复习
3. 今日待复习 / 今日批次 / 日历复习计数，按固定间隔批次计算：
   - 0,1,3,7,14,30 天前录入的新词

使用方式：
1. 把整个文件夹里的所有文件上传到 GitHub 仓库根目录
2. GitHub Pages 从 main 分支 /(root) 发布
3. 用 iPhone Safari 打开网址
4. 添加到主屏幕

建议：
- 更新后如果主屏幕还是旧内容，先在 Safari 打开一次网址刷新
- 还不行就删除主屏幕图标后重新添加
- 定期导出 JSON 备份
